# Projeto Casa de Câmbio

Neste projeto foi desenvolvida uma aplicação que realiza busca da taxa de conversão de determinada moeda através da API [__ExchangeRate-API__](https://www.exchangerate-api.com/docs/overview).

## Tecnologias

  - Vite
  - HTML
  - CSS
  - JavaScript

## Como executar

Clone o projeto e acesse a pasta do mesmo.

```bash
$ git clone git@github.com:Lucas-Almeida-SD/Casa_de_cambio.git

$ cd Casa_de_cambio
```

Para iniciá-lo, siga os passos abaixo:

```bash
# instalar as dependências
$ npm install

# iniciar a aplicação
$ npm run dev
```

Lembrando que para a aplicação funcionar corretamente, você deverá renomear o arquivo __.env.example__ para __.env__ e você deve inserir sua `API KEY` nesse arquivo como valor da variável __API_KEY__. Você conseguirá a sua `API KEY` no site oficial [__ExchangeRate-API__](https://www.exchangerate-api.com/docs/overview) após criar sua conta gratuitamente.
